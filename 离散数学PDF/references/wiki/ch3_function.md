# 第3章 函数

## 3.1 函数的定义
- **函数F**：二元关系，且∀x∈domF存在唯一的y使xFy成立
- **从A到B的函数 f:A→B**：domf=A, ranf⊆B
- **函数集合 Bᴬ** = {f|f:A→B}, |Bᴬ| = |B|^|A|

## 3.2 单射、满射、双射
- **单射**：∀x₁≠x₂⇒f(x₁)≠f(x₂)，即不同x映射到不同y
- **满射**：ranf=B，即每个B中元素都是某x的像
- **双射**：既是单射又是满射

## 3.3 函数的复合与反函数
- **复合 (f∘g)(x) = f(g(x))**
- **反函数**：双射才有反函数f⁻¹
- f⁻¹∘f = IA, f∘f⁻¹ = IB

## 3.4 集合的基数
- **等势**：存在双射⇒基数相等
- **可数集**：与ℕ等势的集合
- **康托尔定理**：|P(A)| > |A|
