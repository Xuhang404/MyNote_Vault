# 1.2集合的运算

设$A,B$均为集合,$A$与$B$的并集$A\cup B$,交集$A\cap B$,$B与A$的相对补集$A - B$分别定义如下( $\lor$和$\land$分别表示自然语言"或"和"且")

$$\begin{align}
    A\cup B = \{ x|x \in A \lor x \in B \}\\
	A\cap B = \{ x|x \in A \land x \in B \}\\
	A - B = \{ x|x \in A \land x \not\in B \}\\
\end{align}
$$
若两个集合的交集为$\emptyset$,则称这两个集合为**不交**的

两个集合的并和交运算可以推广到$n$个集合的并和交

$$
	\begin{align}
	A_{1}\cup A_{2}\cup\dots \cup A_{n} = \{ x|x \in A_{1}\lor x \in A_{2}\lor\dots \lor x \in A_{n} \}\ \\
	A_{1}\cap A_{2}\cap\dots \cap A_{n} = \{ x|x \in A_{1} \land x \in A_{2} \land \dots \land x \in A_{n} \}
	\end{align}
$$
上述的并和交可以分别简记为$\cup ^{n}_{i=1}A_{i}$和$\cap ^{n}_{i=1}A_{i}$

## 对称差和绝对补集

1. 对称差

设$A,B$为集合,$A,B$的对称差$A\oplus B$定义为
$$
A \oplus B = (A-B)\cup (B-A)
$$
例如:若$A=\{ a,b,c \},B=\{ b,d \}$则$A\oplus B = \{ a,c,d \}$

>[!注意]
>对称差可以理解为非公共元素得到的集合

2. 绝对补集

在给定全集$E$后,$E$的子集$A$的绝对补集$\sim A$定义为
$$
	\sim A = E-A=\{ x|x \in E\land x \not\in A \}
$$

例如,若$E = \{ a,b,c,d \},A=\{ a,b,c \}$则,$\sim A = \{ d \}$
