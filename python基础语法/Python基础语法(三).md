# 一 输入

 `input()`的功能就是获取键盘输入的数据
 
```python
# input()

name = input("请输入您的姓名:")
print(f"欢迎您,{name}")

```

>[!注意]
>无论键盘中输入什么样的数据,最终得到的都是字符串类型

---
#例题 

**银行取款:小智的银行卡中有10000元,请根据输入的金额执行取钱操作,完毕后展示余额

```python
total =  10000

num = input("请输入取款金额:")
# num从键盘接收为字符串类型,需要转换为int类型
print(f"您取款{num}元,剩余{total - int(num)}元")
```

# 二 运算符

## 算术运算符

| 运算符 | 描述  | 举例说明              |
| --- | --- | ----------------- |
| +   | 加法  |                   |
| -   | 减法  |                   |
| *   | 乘法  |                   |
| /   | 除法  | 结果为小数             |
| //  | 整除法 | 结果为整数,舍弃小数部分,向下取整 |
| %   | 取模  |                   |
| **  | 幂指数 |                   |

---

#例题 

输入两个数x和y,计算x + y 和 x - y的结果并输出

```python
x = input("请输入x的值")
y = input("请输入y的值")

print(f"x+y={float(x)+float(y)}")
print(f"x-y={float(x)-float(y)}")//注意类型转换
```

## 赋值运算符

| 运算符  | 描述                          |
| ---- | --------------------------- |
| =    | 赋值运算符                       |
| +=   | num += 2 <==> num = num + 2 |
| -=   | num -= 2 <==> num = num - 2 |
| \*=  | 同理                          |
| /=   | 同理                          |
| %=   | 同理                          |
| //=  | 同理                          |
| \**= | 同理                          |

---

## 比较运算符(关系运算符)

比较运算符用于比较两个值之间的关系,会计算运算符两边的表达式,然后返回一个**布尔值**作为结果

| 运算符 | 描述   |
| --- | ---- |
| ==  | 等于   |
| !=  | 不等于  |
| >   | 大于   |
| >=  | 大于等于 |
| <   | 小于   |
| <=  | 小于等于 |

---

## 逻辑运算符

逻辑运算符连接多个表达式(值为真或假),并返回一个布尔结果

| 运算符 | 描述        |
| --- | --------- |
| and | 逻辑与(&&\)  |
| or  | 逻辑或(\|\|) |
| not | 逻辑非(!)    |
