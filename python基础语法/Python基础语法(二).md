# 一 字符串

## ***字符串的三种定义方式

```python
# 双引号定义
s1 = "hello"

# 单引号定义
s2 = 'hello'

# 三引号定义(多行字符串)
s3 = """
hello
world
"""

```

---

## 转义字符

转义字符的写法:在字符前加入反斜线\

常见转义字符:

| 转义字符 | 名称  | 作用         |
| ---- | --- | ---------- |
| \\'  | 单引号 | 表示单引号      |
| \\"  | 双引号 | 表示双引号      |
| \n   | 换行符 | 换行         |
| \t   | 制表符 | 缩进一个tab的大小 |

---

## 字符串拼接

python可以将字符串直接用加号+拼接

>[!说明]
>+号可以用来拼接两个字符串,但是无法将非字符串与字符串进行拼接,需要将非字符串类型转换为字符串类型

将非字符串类型转换为字符串类型的方法:

`str(数字)

例如:
```python

age = 18

msg = "我今年" + str(age) + "岁"

print(msg)

```

输出内容
```bash
我今年18.5岁
```

显然,字符串拼接存在以下问题:
- 拼接繁琐
- 破坏字符串完整性
- 需要进行类型转换

**字符串格式化**可以解决这个问题

---

## 字符串格式化

### 方式一 


通过**百分号%占位符**的形式完成字符串和字面量的快速拼接

```python
str = Xuhang

print("大家好,我是%s" % str)
```

```python
s1 = "人生苦短"
s2 = "我要学计算机"

print("我说:%s,%s"  %(s1,s2))
```

>[!注意]
>前面有多少个占位符%s,后面就要有多少个变量或数据


### 方式二(推荐)

也可以使用`f"内容{变量名/表达式}"`的方式来进行字符串的格式化

```python
name = "涛哥"

print(f"大家好,我是{name}")
```
