# 第1章 集合

## 1.1 集合的基本概念
- **集合**：把一些事物汇集到一起组成一个整体，这些事物称为集合的**元素**
- **三要素**：确定性、互异性、无序性
- **表示法**：列举法 `A={a,b,c}`、描述法 `A={x|x具有性质P}`
- **常见的集合**：ℕ自然数集、ℤ整数集、ℚ有理数集、ℝ实数集、ℂ复数集
- **属于关系**：`∈`（属于）、`∉`（不属于）

## 1.2 子集与相等
- **子集**：`B⊆A` 当且仅当 ∀x(x∈B→x∈A)
- **真子集**：`B⊂A` 当且仅当 B⊆A 且 B≠A
- **集合相等**：`A=B` 当且仅当 A⊆B 且 B⊆A
- **空集 ∅**：不含任何元素，是一切集合的子集，且唯一
- **注意**：∅≠{∅}（空集 ≠ 以空集为元素的集合）
- **幂集 P(A)**：A全体子集构成的集合，|P(A)|=2ⁿ（n为|A|）
- **全集 E**：所涉及的所有集合都是其子集

## 1.3 集合的运算
- **并 A∪B** = {x|x∈A∨x∈B}
- **交 A∩B** = {x|x∈A∧x∈B}\n- **相对补 A−B** = {x|x∈A∧x∉B}
- **对称差 A⊕B** = (A−B)∪(B−A)
- **绝对补 ~A** = E−A

## 1.4 运算性质（重要算律）
| 算律 | 公式 |
|:---|:---|
| 幂等律 | A∪A=A, A∩A=A |
| 结合律 | (A∪B)∪C=A∪(B∪C), (A∩B)∩C=A∩(B∩C) |
| 交换律 | A∪B=B∪A, A∩B=B∩A |
| 分配律 | A∪(B∩C)=(A∪B)∩(A∪C), A∩(B∪C)=(A∩B)∪(A∩C) |
| 同一律 | A∪∅=A, A∩E=A |
| 零律 | A∪E=E, A∩∅=∅ |
| 排中律 | A∪~A=E |
| 矛盾律 | A∩~A=∅ |
| 吸收律 | A∪(A∩B)=A, A∩(A∪B)=A |
| 德摩根律 | ~(B∪C)=~B∩~C, ~(B∩C)=~B∪~C |
| 双重否定 | ~~A=A |

## 1.5 包含排斥原理（容斥原理）
- |A₁∪A₂∪⋯∪An| = Σ|Ai| − Σ|Ai∩Aj| + Σ|Ai∩Aj∩Ak| − ⋯ + (−1)ⁿ⁻¹|A₁∩A₂∩⋯∩An|
- 补形式：|Ā₁∩Ā₂∩⋯∩Ān| = |E| − Σ|Ai| + Σ|Ai∩Aj| − ⋯ + (−1)ⁿ|A₁∩A₂∩⋯∩An|
