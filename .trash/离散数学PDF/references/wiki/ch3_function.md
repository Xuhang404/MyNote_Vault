## 3.1 函数的定义与性质

### 3.1.1 函数的定义
- **函数F**：设F为二元关系，若∀x∈domF都存在**唯一**的y∈ranF使xFy成立，则称F为**函数**
- 记法：若⟨x,y⟩∈F，则记作 y = F(x)，称y为F在x处的**值**

**例 3.1.1**：
- F₁ = {⟨a₁,b₁⟩, ⟨a₂,b₂⟩, ⟨a₃,b₂⟩} → 是函数（每个x对应唯一y）
- F₂ = {⟨a₁,b₁⟩, ⟨a₁,b₂⟩} → **不是**函数（a₁对应两个不同的b）

### 3.1.2 函数相等
- 函数相等⇔两者是同一个集合
- 等价条件：① domF = domG ② ∀x∈domF有 F(x)=G(x)

### 3.1.3 从A到B的函数
- **f: A→B**：f是函数，且 domf=A，ranf⊆B
- A称为**定义域**，B称为**陪域**，ranf称为**值域**

### 3.1.4 函数集合 Bᴬ
- **Bᴬ** = { f | f: A→B }，即所有从A到B的函数构成的集合
- **基数公式**：若|A|=m, |B|=n, m,n>0，则 |Bᴬ| = n^m
- 例：A={1,2,3}, B={a,b}，则 |Bᴬ| = 2³ = 8

**空集特殊情况**：
| A | B | Bᴬ |
|:---|:---|:---|
| ∅ | ∅ | {∅}（唯一函数）|
| ∅ | ≠∅ | {∅}（唯一函数）|
| ≠∅ | ∅ | ∅（无函数）|

### 3.1.5 函数的像与原像
**定义 3.1.5**：设 f: A→B, A₁⊆A, B₁⊆B

① **像 f(A₁)** = { f(x) | x∈A₁ } ⊆ B
- 当 A₁=A 时，f(A) 称为**函数的像**
- 注意：**函数值** f(x)∈B，而**像** f(A₁)⊆B

② **原像 f⁻¹(B₁)** = { x | x∈A, f(x)∈B₁ } ⊆ A

**重要性质**：一般地 f⁻¹(f(A₁)) ≠ A₁，但 A₁ ⊆ f⁻¹(f(A₁))
- 例：f: {1,2,3}→{0,1}, f(1)=f(2)=0, f(3)=1
- A₁={1} ⇒ f⁻¹(f(A₁)) = f⁻¹({0}) = {1,2}，A₁ ⊂ f⁻¹(f(A₁))

**例 3.1.3**：f: ℕ→ℕ, f(n) = { n/2 (n偶), n+1 (n奇) }
- A={0,1}, B={2}
- f(A) = {f(0), f(1)} = {0, 2}
- f⁻¹(B) = f⁻¹({2}) = {1, 4}

### 3.1.6 单射、满射、双射
**定义 3.1.6** 设 f: A→B

| 类型 | 定义 | 等价条件 | \|A\|与\|B\| |
|:---|:---|:---|:---|
| **满射** | ranf = B | ∀y∈B, ∃x∈A 使 f(x)=y | \|A\| ≥ \|B\| |
| **单射** | 不同x映射到不同y | ∀x₁≠x₂⇒f(x₁)≠f(x₂) 或 f(x₁)=f(x₂)⇒x₁=x₂ | \|A\| ≤ \|B\| |
| **双射** | 既是满射又是单射 | 一一对应 | \|A\| = \|B\| |

**判断举例（例3.1.4）**：
| 函数 | 单射 | 满射 | 原因 |
|:---|:---:|:---:|:---|
| f(x)=-x²+2x-1 (ℝ→ℝ) | ✗ | ✗ | 抛物线，最大值0，非单调 |
| f(x)=lnx (ℤ⁺→ℝ) | ✓ | ✗ | 单调递增，但ran⊂ℝ |
| f(x)=⌊x⌋ (ℝ→ℤ) | ✗ | ✓ | 取整函数，满射但f(1.5)=f(1.2)=1 |
| f(x)=2x+1 (ℝ→ℝ) | ✓ | ✓ | **双射**，单调且ran=ℝ |
| f(x)=x/(x²+1) (ℝ⁺→ℝ⁺) | ✗ | ✗ | 在x=1取极大值1/2，非单调 |

### 3.1.7 特征函数
- A'⊆A，A'的特征函数 χ_A': A→{0,1}
- χ_A'(x) = 1 若 x∈A'，= 0 若 x∈A−A'

## 3.2 函数的复合与反函数

### 3.2.1 函数复合定义
**定理 3.2.1**：设f,g是函数，则g∘f也是函数，且：
- dom(g∘f) = { x | x∈domf, f(x)∈domg }
- ∀x∈dom(g∘f)有 (g∘f)(x) = g(f(x))

**推论 3.2.1（结合律）**：(f∘g)∘h = f∘(g∘h)

**推论 3.2.2**：若 f:A→B, g:B→C，则 g∘f: A→C
- dom(g∘f) = A, ran(g∘f) ⊆ C

**结合律证明梗概**：从关系右复合的定理可直接推出

### 3.2.2 复合保持单射/满射/双射
**定理 3.2.2**：设 f:A→B, g:B→C

① f和g都是**满射** ⇒ g∘f是**满射**
② f和g都是**单射** ⇒ g∘f是**单射**
③ f和g都是**双射** ⇒ g∘f是**双射**

**证明思路**：
- 满射：∀c∈C，由g满射∃b∈B使g(b)=c；由f满射∃a∈A使f(a)=b ⇒ g(f(a))=c
- 单射：若g(f(a₁))=g(f(a₂))，由g单射得f(a₁)=f(a₂)，由f单射得a₁=a₂

### 3.2.3 与恒等函数的复合
**定理 3.2.3**：设 f:A→B，则 f = f∘I_A = I_B∘f

### 3.2.4 复合的逆向命题
**定理 3.2.6**：设 f:A→B, g:B→C

① g∘f是**单射** ⇒ f是**单射**（g不一定单射）
② g∘f是**满射** ⇒ g是**满射**（f不一定满射）
③ g∘f是**双射** ⇒ f是单射, g是满射

**反例**：
- 单射反例：A={a₁,a₂,a₃}, B={b₁,b₂,b₃,b₄}, C={c₁,c₂,c₃}
  - f = {⟨a₁,b₁⟩,⟨a₂,b₂⟩,⟨a₃,b₃⟩}（单射）
  - g = {⟨b₁,c₁⟩,⟨b₂,c₂⟩,⟨b₃,c₃⟩,⟨b₄,c₃⟩}（非单射）
  - g∘f = {⟨a₁,c₁⟩,⟨a₂,c₂⟩,⟨a₃,c₃⟩}（单射）→ f单射但g非单射
- 满射反例：A={a₁,a₂,a₃}, B={b₁,b₂,b₃}, C={c₁,c₂}
  - f = {⟨a₁,b₁⟩,⟨a₂,b₂⟩,⟨a₃,b₂⟩}（非满射）
  - g = {⟨b₁,c₁⟩,⟨b₂,c₂⟩,⟨b₃,c₂⟩}（满射）
  - g∘f = {⟨a₁,c₁⟩,⟨a₂,c₂⟩,⟨a₃,c₂⟩}（满射）→ g满射但f非满射

### 3.2.5 反函数
- 一般函数的逆f⁻¹只是二元关系，**不一定是函数**
- 例：f={⟨a₁,b₁⟩,⟨a₂,b₁⟩} ⇒ f⁻¹={⟨b₁,a₁⟩,⟨b₁,a₂⟩}不是函数（b₁对应两个值）

**引理**：单射f:A→B ⇒ f⁻¹是函数，是 ranf→A 的双射
- 但f⁻¹不一定是 B→A 的函数（当f非满射时，B−ranf中的元素无对应值）

**定理 3.2.4**：f:A→B是**双射** ⇒ f⁻¹:B→A也是**双射**
- 此时称f⁻¹为f的**反函数**

**定理 3.2.5**：若f:A→B是双射，则
- f⁻¹∘f = I_A
- f∘f⁻¹ = I_B

## 3.3 双射函数与集合的基数

### 3.3.1 等势的概念
**定义 3.3.1**：若存在从A到B的**双射**函数，则称A和B**等势**，记作 A≈B

**等势的性质（定理3.3.1）**：
① 自反：A≈A
② 对称：若A≈B则B≈A
③ 传递：若A≈B且B≈C则A≈C

### 3.3.2 等势的重要例子

**(1) ℤ ≈ ℕ**（整数集与自然数集等势）
构造双射 f: ℤ→ℕ:
f(n) = { 2n, n≥0; −2n−1, n<0 }
即：0→0, 1→2, −1→1, 2→4, −2→3, ...

**(2) ℕ×ℕ ≈ ℕ**
对角线枚举法（康托尔配对函数）：
⟨0,0⟩→0, ⟨0,1⟩→1, ⟨1,0⟩→2, ⟨0,2⟩→3, ⟨1,1⟩→4, ⟨2,0⟩→5, ...
公式：f(m,n) = (m+n+1)(m+n)/2 + m

**(3) ℕ ≈ ℚ**（有理数集可数）
用二维表列出所有分数p/q（q>0），按对角线遍历并跳过重复的有理数

**(4) (0,1) ≈ ℝ**
构造双射 f: (0,1)→ℝ, f(x) = tan((2x−1)π/2)
（将(0,1)拉伸到整个实数轴）

**[0,1] ≈ (0,1)**（康托尔移位法）：
将[0,1]中的0→1/2, 1→1/4, 1/2→1/8, 1/2ⁿ→1/2ⁿ⁺²，其余点保持不变

**(5) P(A) ≈ {0,1}ᴬ**（幂集与特征函数集等势）
构造 Φ: P(A)→{0,1}ᴬ, Φ(A') = χ_A'（A'的特征函数）
- 单射：不同子集的特征函数不同
- 满射：任意f:A→{0,1}定义A'={x|f(x)=1}，则χ_A' = f

### 3.3.3 康托尔定理
**定理 3.3.2**：
① ℕ ≉ ℝ（自然数与实数不等势）
② 对任意集合A，都有 A ≉ P(A)

**(1) ℕ≉ℝ的对角线证明**：
假设存在双射 f:ℕ→[0,1]，将[0,1]中所有实数排成一列：
r₀=0.a₀₀a₀₁a₀₂..., r₁=0.a₁₀a₁₁a₁₂..., r₂=0.a₂₀a₂₁a₂₂..., ...
构造新数 r* = 0.c₀c₁c₂... 其中 cₙ≠aₙₙ 且 cₙ≠9
则 r*∈[0,1] 但与每个rₙ至少一位不同 ⇒ r*不在列表中 ⇒ 矛盾

**(2) A≉P(A)的证明**：
设任意 f:A→P(A)，构造 B = {x∈A | x∉f(x)} ⊆ A
则对任意x∈A：x∈B ⇔ x∉f(x)，所以 B≠f(x)
⇒ f不是满射，故不存在从A到P(A)的双射

### 3.3.4 优势
**定义 3.3.2**：
① 若存在从A到B的**单射**，则称**B优势于A**，记作 A≼·B
② 若 A≼·B 且 A≉B，则称**B真优势于A**，记作 A≺·B

**定理 3.3.3（伯恩斯坦定理）**：
若 A≼·B 且 B≼·A，则 A≈B

应用示例：用伯恩斯坦定理证明 {0,1}ℕ ≈ [0,1)
- 构造 [0,1)→{0,1}ℕ 的单射（二进制展开）
- 构造 {0,1}ℕ→[0,1) 的单射（十进制展开避免进位问题）

### 3.3.5 有穷集与无穷集
- **有穷集**：与某个 ℕₙ (n∈ℕ) 等势的集合（其中 ℕ₀=∅, ℕₙ={0,1,...,n−1}）
- **无穷集**：不是有穷集的集合
- ℕ和ℝ都是无穷集（无法与任何ℕₙ等势）

### 3.3.6 基数
- **基数**：集合的"大小"，集合等势时基数相同
- **可数集**：与ℕ等势的集合（基数为ℵ₀，读作"阿列夫零"）
- 重要结论：
  - ℕ≈ℤ≈ℚ≈ℤ×ℤ （基数 ℵ₀）
  - ℝ≈[a,b]≈(a,b)≈{0,1}ℕ≈P(ℕ) （基数 ℵ）
  - ℕ ≺· ℝ（真优势）
  - A ≺· P(A)（康托尔定理）
