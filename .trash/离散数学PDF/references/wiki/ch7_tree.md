# 第7章 树

## 7.1 无向树
- **定义**：连通无回路的无向图
- **性质等价命题**：
  1. G是树（连通无回路）
  2. 任意两顶点有唯一路径
  3. 无回路且m=n−1
  4. 连通且m=n−1
  5. 连通且每条边都是桥
- **树叶**：悬挂顶点（度=1）
- **分支点**：度≥2的顶点

## 7.2 生成树
- **生成树**：包含所有顶点的树形子图
- **最小生成树**：带权最小的生成树
- **Kruskal算法**：每次选权最小且不构成回路的边

## 7.3 根树
- **有向树**：恰有一个入度为0的根
- **m叉树**：每个分支点最多m个儿子
- **哈夫曼树**：加权路径长度最小的最优二叉树
- **前缀码**：用哈夫曼树构造
