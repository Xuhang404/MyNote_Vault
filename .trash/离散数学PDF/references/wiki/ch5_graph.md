# 第5章 图的基本概念

---

## 5.1 图的定义及运算

### 5.1.1 无向图与有向图
- **无向图 G=<V,E>**：V是非空顶点集，E是V&V的无序积的有穷多重子集
  - 无序积：V&V = { {a,b} | a∈V, b∈V }，记(a,b)且(a,b)=(b,a)
- **有向图 D=<V,E>**：E是V×V的有穷多重子集
  - 有序对 <a,b>，a为始点，b为终点

### 基本术语
- **n阶图**：有n个顶点的图
- **零图**：无边图。n阶零图记作Nn
- **平凡图**：1阶零图N1
- **空图**：顶点集为空（图运算中可能出现），记作∅
- **标定图**：顶点和边都指定了符号；否则为**非标定图**
- **基图**：将有向图各边改成无向边后所得的无向图

### 关联与相邻
- 边与顶点的关系叫**关联**，环与顶点关联次数为2
- 顶点间有边连接叫**相邻**，边有公共端点叫**相邻**

**无向图**：邻域 N(v)={u|(v,u)∈E, u≠v}，闭邻域 N̄(v)=N(v)∪{v}，关联集 I(v)

**有向图**：后继元集 Γ⁺(v)={u|<v,u>∈E, u≠v}，先驱元集 Γ⁻(v)={u|<u,v>∈E, u≠v}
邻域 Γ(v)=Γ⁺(v)∪Γ⁻(v)，闭邻域 Γ̄(v)=Γ(v)∪{v}

### 特殊图
- **平行边**：关联同一对顶点的多条边（有向图中方向也必须相同）
- **重数**：平行边的条数
- **多重图**：含平行边的图
- **简单图**：既无平行边也无环的图

### 子图与母图
- **子图**：V'⊆V且E'⊆E，记作G'⊆G
- **真子图**：V'⊂V或E'⊂E
- **生成子图**：V'=V（顶点集不变）
- **导出子图**：V₁导出的子图G[V₁]（取V₁和两端都在V₁中的所有边）
  E₁导出的子图G[E₁]（取E₁和E₁中边关联的所有顶点）

### 图的运算
- **删除边** G-e 或 G-E'；**删除顶点** G-v 或 G-V'（连边一起删）
- **收缩边** G\e：删除边e后，将e的两个端点合并为一个新顶点
- **加新边** G∪(u,v) 或 G+(u,v)
- **并图** G₁∪G₂、**差图** G₁−G₂、**交图** G₁∩G₂、**环和** G₁⊕G₂=(G₁∪G₂)−(G₁∩G₂)

## 5.2 度数、通路与回路

### 5.2.1 度数定义
- **无向图** deg(v)=以v为端点的边数（环算2度）
- **有向图**：出度deg⁺(v)=以v为始点的边数，入度deg⁻(v)=以v为终点的边数
  deg(v)=deg⁺(v)+deg⁻(v)（环贡献1出度+1入度）
- **最大度** ∆=max{deg(v)}，**最小度** δ=min{deg(v)}
- **悬挂顶点**：度数为1的顶点，其边为**悬挂边**
- **奇度顶点/偶度顶点**

### 5.2.2 握手定理
**定理 5.2.1（握手定理）**：在任何无向图中，所有顶点度数之和等于边数的2倍
- Σ deg(v) = 2|E|

**定理 5.2.2（有向图握手定理）**：
- Σ deg⁺(v) = Σ deg⁻(v) = |E|
- 所有顶点度数之和 = 2|E|

**推论 5.2.1**：任何图中，奇度顶点的个数是**偶数**

### 5.2.3 补图与自补图
- **补图** Ḡ：以V为顶点集，{ (u,v) | u≠v, (u,v)∉E } 为边集的图
- **自补图**：G ≅ Ḡ（图与其补图同构）
- 自补图的性质：边数 m = n(n-1)/4，所以 n=4k 或 n=4k+1
- 例：n=4时 (边数3)，n=5时 (边数5)

### 5.2.4 通路与回路
**定义 5.2.6**：Γ = v₀e₁v₁e₂v₂...eₖvₖ 为从v₀到vₖ的通路，k为长度

| 类型 | 边 | 顶点(首尾除外) | 首尾 |
|:---|:---|:---|:---:|
| **通路** | 可重复 | 可重复 | 可不相同 |
| **回路** | 可重复 | 可重复 | 相同 |
| **简单通路** | **不重复** | 可重复 | 可不相同 |
| **简单回路** | **不重复** | 可重复 | 相同 |
| **初级通路/路径** | **不重复** | **不重复** | 可不相同 |
| **初级回路/圈** | **不重复** | **不重复** | 相同 |

- **奇圈**：长度为奇数的圈；**偶圈**：长度为偶数的圈
- 初级通路(回路)必是简单通路(回路)，反之不成立

**定理 5.2.5**：n阶图中，若从u到v存在通路(u≠v)，则存在长度≤n-1的通路
**推论**：n阶图中，若从u到v存在通路，则存在长度≤n-1的**初级通路**

## 5.3 图的连通性

### 5.3.1 连通与距离
- **连通**：u~v表示u到v存在通路。规定∀v, v~v
- **连通图**：任意两点都连通的图
- **连通关系** ~ 是V上的等价关系
- **连通分支**：每个等价类导出的子图。p(G)=连通分支数
  - 连通图p(G)=1，n阶零图p(G)=n
- **短程线**：u到v长度最短的通路
- **距离** d(u,v)=短程线长度（不连通时d(u,v)=+∞）
  - 性质：非负性、对称性、三角不等式

### 5.3.2 割集与连通度
- **点割集** V'：删除V'后分支数增加，但删除V'的任何真子集分支数不变
  - **割点**：{v}是点割集
- **边割集** E'：删除E'后分支数增加，删除E'的任何真子集分支数不变
  - **桥(割边)**：{e}是边割集

- **点连通度** κ(G)（非完全连通图）= min{|V'| | V'是点割集}
  - 完全图 Kₙ 的 κ(Kₙ)=n-1，非连通图 κ=0
  - **k-连通图**：κ(G)≥k（删除任何k-1个顶点后仍然连通）
  
- **边连通度** λ(G)（连通图）= min{|E'| | E'是边割集}
  - 非连通图 λ=0，非平凡树 λ=1
  - **k边-连通图**：λ(G)≥k（删除任何k-1条边后仍然连通）

## 5.4 图的矩阵表示

### 5.4.1 关联矩阵
- **无向图关联矩阵** M(G) = (mᵢⱼ)ₙₓₘ
  - mᵢⱼ = vᵢ与eⱼ的关联次数(0,1,2)
  - 性质：每列和为2；第i行和为deg(vᵢ)；所有元素之和=2|E|（握手定理）
- **有向图关联矩阵** M(D)（无环）
  - mᵢⱼ = 1 (vᵢ是始点), -1 (vᵢ是终点), 0 (不关联)
  - 性质：每列恰有一个+1和一个-1

### 5.4.2 邻接矩阵
- **邻接矩阵** A(D) = (aᵢⱼ)ₙₓₙ，aᵢⱼ = vᵢ邻接到vⱼ的边数
- **性质**：行和为出度，列和为入度，所有元素之和=|E|

**核心定理 5.4.1**：A^l 中元素 aᵢⱼ^(l) = vᵢ到vⱼ长度为 l 的通路数
- aᵢᵢ^(l) = vᵢ到自身长度为 l 的回路数
- 所有元素之和 = 长度为 l 的通路（含回路）总数

**推论**：Bₖ = A + A² + ... + Aᵏ，其元素之和 = 长度≤k的通路总数

### 5.4.3 可达矩阵
- **可达矩阵** P(D) = (pᵢⱼ)ₙₓₙ
  - pᵢⱼ = 1 (vᵢ可达vⱼ)，0 (不可达)
  - 主对角线元素全为1
  - 可通过邻接矩阵的幂计算：pᵢⱼ=1 ⇔ ∃l使aᵢⱼ^(l)>0
